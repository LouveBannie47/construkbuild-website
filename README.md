# ConstrukBuild - Site Vitrine BTP

Site web moderne et professionnel pour une entreprise de **Bâtiment et Travaux Publics (BTP)** au Cameroun.

Développé avec **React 19**, **Vite** et **Tailwind CSS 4**.

![Aperçu du site](https://via.placeholder.com/800x400?text=ConstrukBuild+Hero) <!-- Tu peux remplacer par un screenshot plus tard -->

## ✨ Fonctionnalités

- Design moderne et responsive
- Carousel hero avec auto-play
- Section partenaires avec défilement automatique
- Présentation de l'entreprise
- Statistiques impactantes
- Portfolio de réalisations avec effet hover
- Blog sectoriel
- FAQ interactive
- Formulaire de contact + drawer pour demande de devis
- Navigation fluide avec détection de section active
- Animations et transitions soignées

## 🛠 Tech Stack

- **Frontend** : React 19 + Vite
- **Styling** : Tailwind CSS v4 + @tailwindcss/vite
- **Icônes** : React Icons
- **Linting** : ESLint + React Hooks rules
- **Build** : Vite (très rapide)

## 🚀 Installation et développement local

```bash
# Cloner le projet
git clone https://github.com/tonusername/construkbuild-website.git

# Aller dans le dossier
cd construkbuild-website

# Installer les dépendances
npm install

# Lancer en mode développement
npm run dev
